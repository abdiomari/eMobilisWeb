# eMobilisWeb — Building Modern Web Applications with HTML and Bootstrap

## Overview
eMobilisWeb is a web development portfolio showcasing Abdi Omari's skills in building modern web applications using HTML and Bootstrap. This project demonstrates Abdi's ability to create visually appealing and user-friendly web interfaces. The goal of this project is to provide a platform for Abdi to showcase his work and collaborate with others in the web development community.

## Tech Stack
* HTML
* Bootstrap

## Features
* Responsive design with Bootstrap's grid system
* Customizable layouts using Bootstrap's pre-built components
* Integration with external images for a visually appealing interface
* Basic table rendering using HTML tables
* Simple HTML pages with Bootstrap styling

## Setup & Installation
To run the project, follow these steps:

1. Clone the repository using Git:
```bash
git clone https://github.com/abdiomari/eMobilisWeb.git
```
2. Navigate to the project directory:
```bash
cd eMobilisWeb
```
3. Open the `hello-bootstrap.html` file in a web browser to view the project:
```bash
open hello-bootstrap.html
```
Alternatively, you can open the file in a code editor and preview it in a browser.

Note: This project does not require any additional setup or dependencies beyond a basic web browser.
